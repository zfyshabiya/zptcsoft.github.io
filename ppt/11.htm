元素概述

有序列表

ordered list, list item。

ol元素

类型取值
分类 流内容-Flow content
如果子元素中至少有一个li，则变成可感知内容Palpable content
元素上下文 需要流内容的地方
内容模型 li或script-supporting elements(script、template)
属性 全局属性
type属性 - 列标记号类型
start属性 - 列表开始数字
reversed属性 - 是否反向计数

li元素

类型取值
分类 无
元素上下文 ol或ul
内容模型 流内容
属性 全局属性
如果是ol的子元素，具有value属性，表示序数值

有序列表示例

<ol>
    <li>明确的目标</li>
    <li>详细的行动计划</li>
    <li>即可采取行动</li>
    <li>修正自己的行为</li>
    <li>坚持不懈</li>
</ol>
  1. 明确的目标
  2. 详细的行动计划
  3. 即可采取行动
  4. 修正自己的行为
  5. 坚持不懈

无序列表

unordered list, list item。

ul元素

类型取值
分类 流内容-Flow content
如果子元素中至少有一个li，则变成可感知内容Palpable content
元素上下文 需要流内容的地方
内容模型 li或script-supporting elements(script、template)
属性 全局属性

无序列表示例

<ul>
    <li>首页</li>
    <li>产品</li>
    <li>新闻</li>
    <li>关于</li>
    <li>联系</li>
</ul>
  • 首页
  • 产品
  • 新闻
  • 关于
  • 联系

定义列表

dl dt dd

dl元素

类型取值
分类 流内容-Flow content
如果子元素中至少有一个dt-dd，则变成可感知内容Palpable content
元素上下文 需要流内容的地方
内容模型 dt和dd组合
属性 全局属性

dt元素

类型取值
分类 无
元素上下文 dl里的dd之前
内容模型 流内容，但不能包含header、footer、节内容(article aside nav section)、标题内容(h1 h2 h3 h4 h5 h6)等
属性 全局属性

dd元素

类型取值
分类 无
元素上下文 dl里的dt之后
内容模型 流内容
属性 全局属性
